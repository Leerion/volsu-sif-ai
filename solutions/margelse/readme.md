# Востановление внутренних температур по кожным

Задача: восстановить поле глубинных температур молочной железы по кожным. Температура у нас непрерывная величина,
поэтому это задача регрессии. Всего в одной МЖ 9 точек измерений, поэтому нужно восстановить 9 температур, 
т. е. это задача множественной регрессии. Для простоты сначала регрессия идёт по здоровым записям - у них не должно быть 
температурных аномалий и, соответственно, задача облегчается.

## Предварительная обработка и анализ датасета:

Используемый датасет чистый. То есть нету пропусков в данных и выбросов. Но было найдено 13 дубликатов, которые были удалены.

На основе результатов корреляционной матрицы Пирсона были сделаны следующие выводы:

Для кожных и внутренних температур максимальная корреляция наблюдается у соответствующих признаков, например, между ir_0 и mv_0 коффициент корреляции равен 0.8. 

Далее, корреляция наблюдается у соседних точек измерения, например, между ir_1, ir_3 и mv_2 корреляция 0.73. 

Также, корреляция наблюдается между соседними кожными температурами, например, у ir_4 с ir_3 и ir_5 корреляция составляет 0.83 и 0.84 соответственно.

## Base Line:

Применении линейной регрессии с обучающими **стандартизированными** признаками ir_0, ..., ir_8 дало стартувую ошибку *MAE* = 0.390 градуса.

## Генерация новых признаков:

Чтобы снизить корреляцию между входными температурами, были добавлены следующие модификации:квадрат, куб, экспонента с отрицательным аргументом, синус, тангенс, логарифм по основанию 2, квадратный корень, а также среднее арифметическое и разницу максимальной и минимальной кожных температур для каждой строки.
Обучение только на модифицированных признаках дало *MAE* = 0.373.

## LASSO:

С помощью GridsearchCV был найден наилучший параметр **alpha = 0.0001**. В качестве обучающих данных были использованны начальные температуры и сгенерированные признаки. Применялась стандартизация.

Результат остался прежним, *MAE* = 0.373. Но, на основе весов модели были исключены высококореллированне признаки или слабые зависимости. `Таким образом, было принято решение оставить следующие признаки: возведение в квадрат, экспонента с отрицательным аргументом, синус, разница температур у МЖ.`

## Полиномиальная регрессия:

Применение полиномиальной линейной регресии ухудшило результат *MAE* = 0.42. Степень полинома бралась равной 2 (наилучший в интервале от 2 до 9). Полиномиальная регрессия приминялась и только к стартовым температурам ir_0, ..., ir_8, и только к сгенерированным, и к их объединению. Во всех вариантах результат хуже 0.365.

## Цепочная линейная регрессия:

Цепочная линейная регрессия показала наилучший результат *MAE* = 0.373 на признаках, отобранных в процессе анализа весов LASSO.

## Дерево решений:

Для дерева решений были подобраны оптимальные параметры глубины дерева, максимального числа признаков для поиска наиболее информативного, минимального числа объектов в листе, равные 10, 23, 50 соответственно. Ошибка *MAE* = 0.44.

При использования модели с подобранными параметрами в комбинации с цепочной регрессией с порядком нахождения внутренних температур [mw_1, mw_2, mw_3, mw_4, mw_5, mw_6, mw_7, mw_8, mw_0] получилось снизить ошибку до *MAE* = 0.42.

## kNN:

Модель ближайших соседей с оптимальными параметрами алгоритма, количества соседей, p, weights равными 'ball_tree', 26, 2, 'distance' соответственно, показала результат *MAE* = 0.391.

## Стекинг моделей:

В стекинг вошли дерево решений и kNN с подобранными параметрами. В качестве референтной функции использовалась линейная регрессия. Всё это было обёрнуто в модель цепочной регресии. *MAE* = 0.398.

## Аппроксимация:

Распределение непрерывной случайной величины напоминает нормальное распределение. На основе этого наблюдения была опробована функция нормального распределения случайной величины для аппраксимации. Так как входные параметры подаются в виде вектора из 9 значений, было принято решение использовать агрегирующую функцию среднего арифметического.

Таким образом, на вход подавалось одно число. Параметры функции подбирались МНК. Такой подход не показал хороших результатов.

На основе замечания о высокой корреляции соответствующих признаков, было принято решение придавать вес текущей кожной температуре, а после агрегировать среднем арифметическим. То есть, если мы предсказываем $mw_i$ температуру, то температуру $ir_i$ домнажаем на коэффициент, а после считаем среднее арифметическое по всем наружным температурам. 

В результате, *MAE* стало принимать значения от 0.425 до 0.535.
